API

This site has an avaiable JSON API that is very easily called.

All requests must be directed toward:

cgi-bin/json.pl

Search

To search the database 3 flags must be set:

  • selector: "search"
  • type: Either GeneID, ProteinProduct, AccessionNumber, ChromosomeLocation
  • query: Your query

For example, a GeneID search for a gene with the Accession number of "1914776"

cgi-bin/json.pl?&selector=search&type=GeneID&query=1914776

The above would return:

{
   "Y12331" : {
      "GeneName" : "1914776",
      "GeneLength" : "2261",
      "ChromosomeLocation" : "q13",
      "ProteinId" : "CAA72999.1",
      "SeqFeat" : [
         "NCS;0:1164",
         "EXON;1165:1197",
         "NCS;1198:2188",
         "EXON;2189:2261",
         "NCS;2262:2261"
      ],
      "ProteinName" : "mitochondrial serine hydroxymethyltransferase"
   }
}

Single

To show a single gene record

  • selector: "single"
  • query: Your query

For example:

cgi-bin/json.pl?&selector=single&query=1914776

Browse

To browse for Protein product names:

  • selector: "browse"
  • query: Your query

For example:

cgi-bin/json.pl?&selector=single&query=a

Restriction Enzymes

To list all the avaliable restriciton enzymes in the database:

  • selector: "res"
  • mode: "GetRES"

For example:

cgi-bin/json.pl?&selector=res&mode=GetRES

Would return:

{
   "BamHI" : "G|GATCC",
   "EcoRI" : "G|AATTC",
   "BsuMI" : "C|TCGAG"
}

EnzCutter

To cut a sequence or specific gene with a restriction enzyme, use EnzCutter

  • selector: "res"
  • mode: "CalcRES"
  • Either, query: "Nucleic acid sequnce"
  • Or, gene: "Gene ID"
cgi-bin/json.pl?selector=res&mode=CalcRES&query=EcoRI%2CBamHI&gene=TestSequence
